Database

Database adalah kumpulan data yang terorganisir. Bayangkan database sebagai lemari arsip di mana Anda menyimpan berbagai dokumen dan informasi penting dengan cara yang terstruktur sehingga mudah ditemukan dan digunakan kembali.

MySQL

MySQL adalah perangkat lunak sistem manajemen basis data (DBMS) yang menggunakan SQL sebagai bahasa untuk mengelola dan mengakses data. Bayangkan MySQL seperti pustakawan yang tahu di mana semua buku berada dan bagaimana cara mengaksesnya.

SQL

SQL adalah Bahasa yang digunakan untuk mengelola dan memanipulasi data di dalam database. SQL seperti bahasa komunikasi yang Anda gunakan untuk berbicara dengan pustakawan (MySQL).

Tabel

Tabel adalah struktur data yang terorganisir yang terdiri dari baris dan kolom. Bayangkan tabel seperti buku yang terdiri dari halaman dan baris dan kolom. Setiap rak (kolom) memiliki kategori tertentu, dan setiap buku (baris) adalah entri data di dalam kategori itu.

Kolom

Kolom adalah Bagian dari tabel yang menyimpan tipe data tertentu. Setiap kolom memiliki nama dan tipe data (misalnya, teks, angka). Kolom seperti label pada rak buku yang menunjukkan jenis buku (judul, penulis, tahun terbit, dll.).

Baris

Baris adalah Setiap entri atau record dalam tabel yang menyimpan data sesuai dengan kolom. Baris seperti buku individu dalam rak buku. Setiap buku memiliki informasi yang sesuai dengan label raknya (judul buku, penulis buku, dll.).

Perintah SQL

Perintah-perintah MySQL untuk mengelola database mencakup beberapa kategori utama: perintah untuk pembuatan dan pengelolaan database, perintah untuk pembuatan dan pengelolaan tabel, dan perintah untuk manipulasi data. Mari kita jelajahi perintah-perintah ini dengan detail dan contoh-contohnya.

Membuat Database

Untuk membuat database baru, gunakan perintah CREATE DATABASE: 

        CREATE DATABASE nama_database;

contohnya 

        CREATE DATABASE perpustakaan;

Melihat Database Yang Tersedia

Untuk melihat daftar semua database yang ada, gunakan perintah SHOW DATABASES: 

        SHOW DATABASES;

Ini akan menampilkan daftar semua database di server MySQL Anda.

Menggunakan Database

Untuk memilih database yang ingin Anda gunakan, gunakan perintah USE: 

        USE nama_database;

Contohnya 

        USE perpustakaan;

Ini akan mengatur perpustakaan sebagai database aktif yang Anda gunakan untuk operasi berikutnya.

Menghapus Database

Untuk menghapus database, gunakan perintah DROP DATABASE: 

        DROP DATABASE nama_database;

contohnya 

        DROP DATABASE perpustakaan;

Ini akan menghapus database perpustakaan dan semua tabel serta data yang ada di dalamnya.
