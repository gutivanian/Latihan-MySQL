Fungsi Flow Control

Di MySQL, fungsi flow control seperti IF, CASE, LOOP, IFNULL, dan penggunaan nested queries dapat digunakan untuk mengatur alur eksekusi pernyataan SQL dan menangani data dengan lebih fleksibel.

IF Statement

Pernyataan IF digunakan untuk mengevaluasi sebuah kondisi dan mengeksekusi blok pernyataan berdasarkan hasil evaluasi tersebut.

Misalkan kita memiliki tabel students: 

        +----+----------+-------+
        | id | name     | score |
        +----+----------+-------+
        | 1  | Alice    | 75    |
        | 2  | Bob      | 50    |
        | 3  | Charlie  | 85    |
        +----+----------+-------+

Kita dapat menggunakan pernyataan IF untuk menampilkan nama siswa dan kategori nilai mereka berdasarkan skor mereka: 

        SELECT 
            id, 
            name, 
            IF(score >= 60, 'Pass', 'Fail') AS result
        FROM students;

Hasilnya 

        +----+----------+--------+
        | id | name     | result |
        +----+----------+--------+
        | 1  | Alice    | Pass   |
        | 2  | Bob      | Fail   |
        | 3  | Charlie  | Pass   |
        +----+----------+--------+

CASE Statement

Pernyataan CASE digunakan untuk mengevaluasi beberapa kondisi dan mengeksekusi blok pernyataan berdasarkan kondisi yang cocok. 

        SELECT 
            id,
            name,
            CASE 
                WHEN score >= 90 THEN 'A'
                WHEN score >= 80 THEN 'B'
                WHEN score >= 70 THEN 'C'
                ELSE 'D'
            END AS grade
        FROM students;

Hasilnya 

        +----+----------+-------+
        | id | name     | grade |
        +----+----------+-------+
        | 1  | Alice    | B     |
        | 2  | Bob      | D     |
        | 3  | Charlie  | A     |
        +----+----------+-------+

LOOP Statement

        Meskipun MySQL tidak memiliki struktur loop seperti bahasa pemrograman umumnya, Anda dapat mencapainya dengan menggunakan stored procedure dan cursor untuk melakukan iteras

IFNULL Function

Fungsi IFNULL digunakan untuk mengembalikan nilai alternatif jika nilai yang diperiksa NULL. 

        SELECT 
            id,
            name,
            IFNULL(email, 'No email provided') AS email
        FROM customers;

Misalkan kita memiliki tabel customers: 

        +----+----------+---------------------+
        | id | name     | email               |
        +----+----------+---------------------+
        | 1  | Alice    | alice@example.com   |
        | 2  | Bob      | NULL                |
        | 3  | Charlie  | charlie@example.com |
        +----+----------+---------------------+

Hasilnya 

        +----+----------+----------------------+
        | id | name     | email                |
        +----+----------+----------------------+
        | 1  | Alice    | alice@example.com    |
        | 2  | Bob      | No email provided    |
        | 3  | Charlie  | charlie@example.com  |
        +----+----------+----------------------+
