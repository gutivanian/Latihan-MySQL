Auto Increment

AUTO_INCREMENT adalah atribut yang digunakan dalam SQL untuk menghasilkan nilai unik otomatis pada kolom tertentu, biasanya digunakan pada kolom PRIMARY KEY atau UNIQUE. Setiap kali baris baru ditambahkan ke tabel, nilai kolom yang memiliki atribut AUTO_INCREMENT akan meningkat secara otomatis.

Setiap baris baru akan mendapatkan nilai yang unik. Nilai akan bertambah secara otomatis berdasarkan nilai maksimum yang ada di kolom tersebut. Sering digunakan bersama dengan PRIMARY KEY untuk memastikan bahwa setiap baris dalam tabel dapat diidentifikasi secara unik.

Misalkan kita memiliki tabel users dengan kolom id yang menggunakan AUTO_INCREMENT untuk primary key:

  1. Membuat Tabel: 
                    CREATE TABLE users (
                    id INT AUTO_INCREMENT PRIMARY KEY,
                    nama VARCHAR(50),
                    umur INT
                );
  2. Menambahkan Data tanpa id: 
                    INSERT INTO users (nama, umur) VALUES ('Alice', 25);
                INSERT INTO users (nama, umur) VALUES ('Bob', 30);
                INSERT INTO users (nama, umur) VALUES ('Charlie', 28);
  3. Hasil Tabel Setelah Penambahan Data: +----+---------+------+ | ID | Nama | Umur | +----+---------+------+ | 1 | Alice | 25 | | 2 | Bob | 30 | | 3 | Charlie | 28 | +----+---------+------+

Tabel terbentuk dengan nilai id yang otomatis bertambah setiap kali data baru dimasukkan.
