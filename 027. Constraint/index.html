Constraint

Constraints di MySQL digunakan untuk menentukan aturan pada data dalam tabel. Mereka membantu menjaga integritas dan konsistensi data dalam database.

NOT NULL

NOT NULL memastikan bahwa kolom tidak dapat memiliki nilai NULL. 

        CREATE TABLE employees (
            id INT NOT NULL,
            name VARCHAR(50) NOT NULL,
            department VARCHAR(50),
            salary DECIMAL(10, 2)
        );

id INT NOT NULL: Kolom id tidak dapat berisi nilai NULL. name VARCHAR(50) NOT NULL: Kolom name tidak dapat berisi nilai NULL.

UNIQUE

UNIQUE memastikan bahwa semua nilai di kolom adalah unik. 

        CREATE TABLE employees (
            id INT NOT NULL,
            email VARCHAR(100) UNIQUE,
            name VARCHAR(50),
            department VARCHAR(50),
            salary DECIMAL(10, 2)
        );

email VARCHAR(100) UNIQUE: Kolom email harus berisi nilai yang unik.

PRIMARY KEY

PRIMARY KEY adalah kombinasi dari NOT NULL dan UNIQUE. Kolom atau kumpulan kolom yang ditetapkan sebagai primary key akan memiliki nilai unik dan tidak boleh NULL. 

        CREATE TABLE employees (
            id INT PRIMARY KEY,
            name VARCHAR(50),
            department VARCHAR(50),
            salary DECIMAL(10, 2)
        );

id INT PRIMARY KEY: Kolom id ditetapkan sebagai primary key.

FOREIGN KEY

FOREIGN KEY digunakan untuk menghubungkan kolom dalam satu tabel dengan primary key di tabel lain. 

        CREATE TABLE departments (
            id INT PRIMARY KEY,
            name VARCHAR(50) NOT NULL
        );

        CREATE TABLE employees (
            id INT PRIMARY KEY,
            name VARCHAR(50),
            department_id INT,
            FOREIGN KEY (department_id) REFERENCES departments(id)
        );

FOREIGN KEY (department_id) REFERENCES departments(id): Kolom department_id di tabel employees adalah foreign key yang merujuk ke kolom id di tabel departments.

CHECK

CHECK memastikan bahwa nilai dalam kolom memenuhi kondisi tertentu. 

        CREATE TABLE employees (
            id INT PRIMARY KEY,
            name VARCHAR(50),
            salary DECIMAL(10, 2),
            CHECK (salary >= 0)
        );

CHECK (salary >= 0): Kolom salary harus memiliki nilai yang lebih besar atau sama dengan 0.

DEFAULT

DEFAULT memberikan nilai default untuk kolom jika tidak ada nilai yang disediakan.

        CREATE TABLE employees (
            id INT PRIMARY KEY,
            name VARCHAR(50),
            department VARCHAR(50) DEFAULT 'General',
            salary DECIMAL(10, 2)
        );

department VARCHAR(50) DEFAULT 'General': Kolom department akan memiliki nilai default 'General' jika tidak ada nilai yang disediakan.
