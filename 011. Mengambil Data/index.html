Mengambil Data

Untuk mengambil data dari sebuah tabel dalam MySQL, Anda menggunakan perintah SELECT. Perintah ini memungkinkan Anda untuk mengambil data dari satu atau lebih tabel, serta memfilter data, mengurutkannya, atau melakukan operasi lainnya.

Mengambil Semua Data dari Tabel

Untuk mengambil semua data dari sebuah tabel, Anda dapat menggunakan perintah SELECT * FROM nama_tabel;. Tanda * digunakan untuk menunjukkan bahwa Anda ingin mengambil semua kolom yang ada dalam tabel. Contoh: 

        SELECT * FROM users;

Perintah di atas akan mengembalikan semua data yang ada di dalam tabel users, termasuk semua kolom yang ada dalam tabel.

Memilih Kolom Tertentu

Jika Anda hanya tertarik dengan beberapa kolom tertentu, Anda dapat menyebutkan nama kolom-kolom tersebut secara eksplisit dalam perintah SELECT. Contoh: 

        SELECT username, email FROM users;

Perintah di atas akan mengambil hanya kolom username dan email dari tabel users.

Menambahkan Kondisi dengan WHERE

Untuk memfilter hasil query berdasarkan kriteria tertentu, Anda dapat menggunakan klausa WHERE. Contoh: 

        SELECT * FROM users
        WHERE registration_date >= '2023-01-01';

Perintah di atas akan mengambil semua data dari tabel users dimana registration_date lebih besar atau sama dengan tanggal '2023-01-01'.

Mengurutkan Hasil dengan ORDER BY

Untuk mengurutkan hasil query berdasarkan satu atau lebih kolom, gunakan klausa ORDER BY. Contoh:

        SELECT * FROM users
        ORDER BY registration_date DESC;

Perintah di atas akan mengambil semua data dari tabel users dan mengurutkannya berdasarkan kolom registration_date secara descending (terbesar ke terkecil).

Mengambil Jumlah Baris dengan LIMIT

Untuk membatasi jumlah baris yang dikembalikan oleh query, Anda dapat menggunakan klausa LIMIT. Contoh: 

        SELECT * FROM users
        LIMIT 10;

Perintah di atas akan mengambil maksimal 10 baris pertama dari tabel users.
