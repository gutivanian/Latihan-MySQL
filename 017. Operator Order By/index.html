Operator Order By

Di MySQL, klausa ORDER BY digunakan untuk mengurutkan hasil query berdasarkan satu atau lebih kolom dalam urutan tertentu, baik secara ascending (default) atau descending. Operator ORDER BY sangat penting karena memungkinkan Anda untuk mengontrol urutan hasil query yang dikembalikan.

Penggunaan Dasar ORDER BY

Pada dasarnya, klausa ORDER BY digunakan dalam pernyataan SELECT untuk mengurutkan hasil query berdasarkan kolom tertentu. Berikut adalah contoh penggunaan dasar ORDER BY: 

        SELECT * FROM users
        ORDER BY username;

Pernyataan di atas akan mengembalikan semua baris dari tabel users, diurutkan secara ascending (secara default) berdasarkan nilai kolom username.

Urutan Descending

Untuk mengurutkan data secara descending (dari besar ke kecil atau Z-A), Anda bisa menambahkan kata kunci DESC setelah nama kolom: 

        SELECT * FROM users
        ORDER BY username DESC;

Perintah ini akan mengurutkan hasil query secara descending berdasarkan nilai kolom username.

Menggunakan Multiple Columns

Anda juga bisa menggunakan ORDER BY dengan multiple columns untuk mengurutkan data berdasarkan lebih dari satu kolom. Misalnya, untuk mengurutkan berdasarkan nama secara ascending dan umur secara descending: 

        SELECT * FROM users
        ORDER BY nama ASC, umur DESC;

Penggunaan dalam Hasil Fungsi atau Ekspresi

Anda dapat menggunakan ORDER BY dalam hasil fungsi atau ekspresi yang dihitung dalam query: 

        SELECT username, YEAR(registration_date) AS tahun_pendaftaran
        FROM users
        ORDER BY tahun_pendaftaran DESC;

Dalam contoh di atas, ORDER BY tahun_pendaftaran DESC mengurutkan hasil query berdasarkan tahun pendaftaran secara descending.
