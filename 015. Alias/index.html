Alias

Di MySQL, Anda dapat menggunakan alias untuk memberikan nama alternatif untuk tabel atau kolom dalam sebuah query. Alias digunakan untuk membuat query lebih mudah dibaca atau untuk mengurangi panjang nama kolom atau tabel yang digunakan secara berulang dalam query yang kompleks.

Menggunakan Alias untuk Kolom

Anda dapat memberikan alias untuk kolom dalam pernyataan SELECT untuk membuat hasil query lebih mudah dipahami atau untuk mengubah nama kolom menjadi sesuatu yang lebih deskriptif: 

        SELECT
            username AS nama_pengguna,
            email AS alamat_email
        FROM
            users;

Di sini, username diubah namanya menjadi nama_pengguna dan email diubah namanya menjadi alamat_email. Penggunaan alias seperti ini membantu meningkatkan keterbacaan query.

Menggunakan Alias untuk Tabel

Anda juga dapat memberikan alias untuk tabel, meskipun tanpa menggunakan JOIN, untuk mempersingkat penulisan nama tabel dalam query: 

        SELECT
            u.username,
            u.email
        FROM
            users AS u;

Dalam contoh di atas, users AS u memberikan alias u untuk tabel users. Ini memungkinkan Anda untuk merujuk ke kolom-kolom dalam tabel users menggunakan alias u dalam seluruh pernyataan SELECT.

Menggunakan Alias dalam Fungsi atau Ekspresi

Alias juga bisa digunakan untuk memberi nama pada hasil dari fungsi atau ekspresi yang dihitung dalam pernyataan SELECT: 

        SELECT
            username,
            YEAR(registration_date) AS tahun_pendaftaran
        FROM
            users;

Di sini, YEAR(registration_date) mengembalikan tahun dari kolom registration_date, dan hasilnya diberi nama tahun_pendaftaran menggunakan alias.
