Mengupdate Data

Untuk mengupdate data dalam tabel MySQL, Anda menggunakan perintah UPDATE. Perintah ini memungkinkan Anda untuk mengubah nilai kolom-kolom tertentu dalam satu atau lebih baris yang sudah ada di dalam tabel. Berikut adalah cara penggunaan dasar perintah UPDATE:

Misalkan kita memiliki tabel users dengan struktur sebagai berikut: 

        CREATE TABLE users (
            id INT AUTO_INCREMENT PRIMARY KEY,
            username VARCHAR(50) NOT NULL,
            email VARCHAR(100) NOT NULL,
            registration_date DATE
        );

Untuk mengupdate data, gunakan perintah UPDATE dengan menentukan tabel yang ingin diupdate, kolom-kolom yang ingin diubah, dan kondisi untuk mengidentifikasi baris yang akan diupdate.

Mengupdate Satu Kolom

        UPDATE users
        SET email = 'new_email@email.com'
        WHERE id = 1;

Perintah di atas akan mengubah nilai kolom email menjadi 'new_email@email.com' untuk baris yang memiliki id = 1.

Mengupdate Beberapa Kolom

        UPDATE users
        SET username = 'new_username', email = 'new_email@email.com'
        WHERE id = 2;

Perintah di atas akan mengubah nilai kolom username menjadi 'new_username' dan email menjadi 'new_email@email.com' untuk baris yang memiliki id = 2.
