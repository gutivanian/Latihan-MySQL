Fungsi String

SQL menyediakan berbagai fungsi string yang dapat digunakan untuk memanipulasi dan mengolah data string di dalam database.

CONCAT()

Menggabungkan dua atau lebih string menjadi satu. 

        SELECT CONCAT('Hello', ' ', 'World') AS Result;

Hasilnya 

        +--------------+
        | Result       |
        +--------------+
        | Hello World  |
        +--------------+

UPPER()

Mengubah semua karakter menjadi huruf kapital. 

        SELECT UPPER('hello') AS Result;

Hasilnya 

        +--------------+
        | Result       |
        +--------------+
        | HELLO        |
        +--------------+

LOWER()

Mengubah semua karakter menjadi huruf kecil. 

        SELECT LOWER('WORLD') AS Result;

Hasilnya 

        +---------+
        | Result  |
        +---------+
        | world   |
        +---------+

LENGTH()

Mengembalikan panjang string. 

        SELECT LENGTH('Hello') AS Result;

Hasilnya 

        +--------+
        | Result |
        +--------+
        | 5      |
        +--------+

SUBSTRING()

Mengambil sebagian string dari posisi tertentu. 

        SELECT SUBSTRING('Hello World', 7) AS Result;

Hasilnya 

        +--------+
        | Result |
        +--------+
        | World  |
        +--------+

REPLACE()

Menggantikan bagian dari string dengan string lain. 

        SELECT REPLACE('Hello World', 'Hello', 'Hi') AS Result;

Hasilnya 

        +-------------+
        | Result      |
        +-------------+
        | Hi World    |
        +-------------+

        +-------------+
