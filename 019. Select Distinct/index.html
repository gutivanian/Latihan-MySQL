Select Distinct

Klausa SELECT DISTINCT digunakan untuk menghapus duplikasi dari hasil query di MySQL. Ketika Anda menggunakan SELECT DISTINCT, hanya baris dengan nilai unik untuk kolom atau kombinasi kolom yang dipilih yang akan dikembalikan dalam hasil query.

Misalkan kita memiliki tabel users dengan data berikut: 

        +----+----------+-------------------+
        | id | username | email             |
        +----+----------+-------------------+
        | 1  | alice    | alice@example.com |
        | 2  | bob      | bob@example.com   |
        | 3  | alice    | alice@another.com |
        | 4  | dave     | dave@example.com  |
        | 5  | eve      | eve@example.com   |
        | 6  | bob      | bob@another.com   |
        | 7  | alice    | alice@example.com |
        +----+----------+-------------------+

Jika kita ingin mendapatkan daftar username unik dari tabel users, kita bisa menggunakan query berikut: 

        SELECT DISTINCT username
        FROM users;

SELECT DISTINCT username: Memilih hanya nilai unik dari kolom username. Query di atas akan menghasilkan daftar username unik: 

        +----------+
        | username |
        +----------+
        | alice    |
        | bob      |
        | dave     |
        | eve      |
        +----------+

Contoh Lain dengan Multiple Columns

Jika kita ingin mendapatkan kombinasi unik dari beberapa kolom, misalnya username dan email, kita bisa menggunakan query berikut: 

        SELECT DISTINCT username, email
        FROM users;

Hasil query ini akan mengembalikan kombinasi unik dari username dan email: 

        +----------+-------------------+
        | username | email             |
        +----------+-------------------+
        | alice    | alice@example.com |
        | bob      | bob@example.com   |
        | alice    | alice@another.com |
        | dave     | dave@example.com  |
        | eve      | eve@example.com   |
        | bob      | bob@another.com   |
        +----------+-------------------+
