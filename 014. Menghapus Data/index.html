Menghapus Data

Untuk menghapus data dari tabel MySQL, Anda menggunakan perintah DELETE. Perintah ini memungkinkan Anda untuk menghapus satu atau lebih baris data yang ada dalam tabel berdasarkan kondisi yang Anda tentukan. Berikut adalah cara penggunaan dasar perintah DELETE:

Misalkan kita memiliki tabel users dengan struktur sebagai berikut: 

        CREATE TABLE users (
            id INT AUTO_INCREMENT PRIMARY KEY,
            username VARCHAR(50) NOT NULL,
            email VARCHAR(100) NOT NULL,
            registration_date DATE
        );

Untuk menghapus data, gunakan perintah DELETE dengan menentukan tabel yang ingin dihapus dari, serta kondisi untuk mengidentifikasi baris yang akan dihapus.

Menghapus Baris dengan Kondisi Tertentu

        DELETE FROM users
        WHERE id = 1;

Perintah di atas akan menghapus baris dari tabel users dimana nilai kolom id sama dengan 1.

Menghapus Semua Data dalam Tabel

        DELETE FROM users;

Perintah di atas akan menghapus semua baris data dari tabel users. Ini akan mengosongkan seluruh tabel, namun struktur tabelnya tetap ada.
