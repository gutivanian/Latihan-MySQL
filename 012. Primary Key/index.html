Primary Key

Primary key (kunci utama) dalam MySQL adalah sebuah kolom atau sekelompok kolom yang unik mengidentifikasi setiap baris dalam sebuah tabel.

Karakteristik Primary Key:

  1. Setiap nilai primary key harus unik di dalam tabel. Tidak boleh ada dua baris dengan nilai primary key yang sama.
  2. Nilai primary key tidak boleh bernilai NULL. Hal ini memastikan bahwa setiap baris dalam tabel memiliki identitas yang jelas.
  3. Secara default, MySQL akan membuat indeks untuk kolom primary key. Hal ini meningkatkan kinerja pencarian dan pengurutan data.

Menetapkan Primary Key:

Primary key bisa ditetapkan saat membuat tabel dengan menambahkan klausa PRIMARY KEY di definisi kolom: 

        CREATE TABLE nama_tabel (
            id INT AUTO_INCREMENT PRIMARY KEY,
            nama_kolom VARCHAR(50),
            ...
        );

Dalam contoh di atas, kolom id ditetapkan sebagai primary key dengan menggunakan PRIMARY KEY dan AUTO_INCREMENT untuk menghasilkan nilai yang unik secara otomatis.

Mengubah atau Menambahkan Primary Key

Anda bisa mengubah primary key sebuah tabel menggunakan perintah ALTER TABLE, atau menambahkannya jika belum ada primary key sebelumnya: 

        ALTER TABLE nama_tabel
        ADD PRIMARY KEY (nama_kolom);

Di sini, nama_kolom adalah nama kolom yang ingin Anda tetapkan sebagai primary key. Dalam sebuah tabel users, primary key sering kali digunakan pada kolom id yang unik untuk setiap pengguna. Ini memastikan setiap pengguna memiliki identifikasi unik, memudahkan untuk mengelola dan mengakses data pengguna secara efisien.
