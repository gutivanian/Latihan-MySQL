Operator Where

Di MySQL, klausa WHERE digunakan untuk menentukan kondisi atau kriteria yang harus dipenuhi oleh baris data yang akan diambil atau dimodifikasi dalam perintah SQL. Klausa WHERE memungkinkan Anda untuk melakukan pencarian, filtering, atau pengelompokan data berdasarkan kondisi tertentu.

Penggunaan Dasar WHERE

Pada dasarnya, klausa WHERE digunakan dalam perintah SELECT, UPDATE, DELETE, dan pernyataan lainnya untuk membatasi baris data yang diproses berdasarkan kondisi tertentu. Contoh sederhana penggunaan WHERE dalam pernyataan SELECT: 

        SELECT * FROM users
        WHERE username = 'john';

Pernyataan di atas akan mengambil semua kolom dari tabel users dimana nilai kolom username sama dengan 'john'

Operator Perbandingan dalam WHERE

Operator perbandingan digunakan untuk membandingkan nilai di dalam kondisi WHERE. Berikut adalah operator-operator perbandingan yang bisa digunakan:

Equal to (=)

Membandingkan apakah nilai di kedua sisi operator sama. 

        SELECT * FROM users
        WHERE username = 'john';

Not equal to (<> or !=)

Membandingkan apakah nilai di kedua sisi operator tidak sama. 

        SELECT * FROM users
        WHERE username <> 'john';

Greater than (>) and Less than (<)

Membandingkan apakah nilai di sebelah kiri lebih besar atau lebih kecil dari nilai di sebelah kanan. 

        SELECT * FROM users
        WHERE registration_date > '2023-01-01';

Greater than or equal to (>=) and Less than or equal to (<=)

Membandingkan apakah nilai di sebelah kiri lebih besar atau sama dengan atau lebih kecil atau sama dengan nilai di sebelah kanan. 

        SELECT * FROM users
        WHERE registration_date >= '2023-01-01';

BETWEEN

Memeriksa apakah nilai berada di antara dua nilai tertentu (inklusif). 

        SELECT * FROM users
        WHERE registration_date BETWEEN '2023-01-01' AND '2023-12-31';

IN

Memeriksa apakah nilai ada dalam daftar nilai tertentu. 

        SELECT * FROM users
        WHERE username IN ('john', 'mary', 'alice');

LIKE

Digunakan untuk mencocokkan pola string menggunakan wildcard (% untuk nol atau lebih karakter dan _ untuk satu karakter). 

        SELECT * FROM users
        WHERE username LIKE 'j%';

IS NULL dan IS NOT NULL

Memeriksa apakah nilai kolom NULL atau tidak NULL. 

        SELECT * FROM users
        WHERE email IS NULL;

Operator Logika dalam WHERE

Operator logika digunakan untuk menggabungkan kondisi dalam klausa WHERE:

AND

Memeriksa apakah dua kondisi bernilai TRUE. 

        SELECT * FROM users
        WHERE username = 'john' AND email = 'john@email.com';

OR

Memeriksa apakah setidaknya salah satu dari dua kondisi bernilai TRUE. 

        SELECT * FROM users
        WHERE username = 'john' OR username = 'mary';

NOT

Membalikkan hasil kondisi. 

        SELECT * FROM users
        WHERE NOT username = 'john';

Kombinasi Kondisi

Anda dapat mengombinasikan operator perbandingan dan logika untuk menciptakan kondisi yang lebih kompleks: 

        SELECT * FROM users
        WHERE (username = 'john' OR username = 'mary')
        AND registration_date >= '2023-01-01';

Pernyataan di atas akan mengambil semua kolom dari tabel users dimana username adalah 'john' atau 'mary' dan registration_date lebih besar atau sama dengan '2023-01-01'.
