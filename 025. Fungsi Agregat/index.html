Fungsi Agregat

Fungsi agregat di MySQL digunakan untuk melakukan perhitungan pada sekumpulan nilai dan mengembalikan nilai tunggal. Fungsi-fungsi agregat umum termasuk SUM, AVG, COUNT, MAX, dan MIN.

SUM

Fungsi SUM digunakan untuk menjumlahkan nilai dari kolom yang dipilih. Misalkan kita memiliki tabel employees: 

        +----+------------+--------+
        | id | department | salary |
        +----+------------+--------+
        | 1  | HR         | 5000   |
        | 2  | IT         | 6000   |
        | 3  | HR         | 4500   |
        | 4  | IT         | 7000   |
        | 5  | Sales      | 5500   |
        +----+------------+--------+

Contoh Kode SQLnya 

        SELECT 
            department,
            SUM(salary) AS total_salary
        FROM employees
        GROUP BY department;

Hasilnya 

        +------------+-------------+
        | department | total_salary|
        +------------+-------------+
        | HR         | 9500        |
        | IT         | 13000       |
        | Sales      | 5500        |
        +------------+-------------+

AVG

Fungsi AVG digunakan untuk menghitung nilai rata-rata dari kolom yang dipilih. 

        SELECT 
            department,
            AVG(salary) AS avg_salary
        FROM employees
        GROUP BY department;

Hasilnya 

        +------------+------------+
        | department | avg_salary |
        +------------+------------+
        | HR         | 4750       |
        | IT         | 6500       |
        | Sales      | 5500       |
        +------------+------------+

COUNT

Fungsi COUNT digunakan untuk menghitung jumlah baris dalam kolom yang dipilih. 

        SELECT 
            department,
            COUNT(*) AS num_employees
        FROM employees
        GROUP BY department;

Hasilnya 

        +------------+--------------+
        | department | num_employees|
        +------------+--------------+
        | HR         | 2            |
        | IT         | 2            |
        | Sales      | 1            |
        +------------+--------------+

MAX

Fungsi MAX digunakan untuk menemukan nilai maksimum dari kolom yang dipilih. 

        SELECT 
            department,
            MAX(salary) AS max_salary
        FROM employees
        GROUP BY department;

Hasilnya 

        +------------+------------+
        | department | max_salary |
        +------------+------------+
        | HR         | 5000       |
        | IT         | 7000       |
        | Sales      | 5500       |
        +------------+------------+

MIN

Fungsi MIN digunakan untuk menemukan nilai minimum dari kolom yang dipilih. 

        SELECT 
            department,
            MIN(salary) AS min_salary
        FROM employees
        GROUP BY department;

Hasilnya 

        +------------+------------+
        | department | min_salary |
        +------------+------------+
        | HR         | 4500       |
        | IT         | 6000       |
        | Sales      | 5500       |
        +------------+------------+

Fungsi Agregat Lain

Nama Deskripsi Contoh Penggunaan
AVG() Mengembalikan nilai rata-rata dari argumen SELECT AVG(salary) FROM employees;
BIT_AND() Mengembalikan bitwise AND SELECT BIT_AND(flags) FROM table_name;
BIT_OR() Mengembalikan bitwise OR SELECT BIT_OR(flags) FROM table_name;
BIT_XOR() Mengembalikan bitwise XOR SELECT BIT_XOR(flags) FROM table_name;
COUNT() Mengembalikan jumlah baris yang dikembalikan SELECT COUNT(*) FROM employees;
COUNT(DISTINCT) Mengembalikan jumlah nilai yang berbeda SELECT COUNT(DISTINCT department) FROM employees;
GROUP_CONCAT() Mengembalikan string yang digabungkan SELECT GROUP_CONCAT(name) FROM employees;
JSON_ARRAYAGG() Mengembalikan set hasil sebagai array JSON tunggal SELECT JSON_ARRAYAGG(name) FROM employees;
JSON_OBJECTAGG() Mengembalikan set hasil sebagai objek JSON tunggal SELECT JSON_OBJECTAGG(id, name) FROM employees;
MAX() Mengembalikan nilai maksimum SELECT MAX(salary) FROM employees;
MIN() Mengembalikan nilai minimum SELECT MIN(salary) FROM employees;
STD() Mengembalikan standar deviasi populasi SELECT STD(salary) FROM employees;
STDDEV() Mengembalikan standar deviasi populasi SELECT STDDEV(salary) FROM employees;
STDDEV_POP() Mengembalikan standar deviasi populasi SELECT STDDEV_POP(salary) FROM employees;
STDDEV_SAMP() Mengembalikan standar deviasi sampel SELECT STDDEV_SAMP(salary) FROM employees;
SUM() Mengembalikan jumlah total SELECT SUM(salary) FROM employees;
VAR_POP() Mengembalikan varians populasi SELECT VAR_POP(salary) FROM employees;
VAR_SAMP() Mengembalikan varians sampel SELECT VAR_SAMP(salary) FROM employees;
VARIANCE() Mengembalikan varians populasi SELECT VARIANCE(salary) FROM employees;
