SQL (Structured Query Language)

SQL (Structured Query Language) adalah bahasa standar yang digunakan untuk mengelola dan memanipulasi basis data dalam sistem manajemen basis data (DBMS). SQL memungkinkan pengguna untuk berinteraksi dengan basis data relasional dengan menggunakan perintah-perintah yang dirancang untuk berbagai operasi seperti penyimpanan, pengambilan, pembaruan, dan penghapusan data.

Perintah dasar SQL:

  1. DDL (Data Definition Language) digunakan untuk mendefinisikan dan mengelola struktur basis data, termasuk tabel, skema, indeks, dan constraints.
  2. DML (Data Manipulation Language) digunakan untuk memanipulasi data dalam tabel, termasuk memasukkan, memperbarui, dan menghapus data.
  3. DCL (Data Control Language) digunakan untuk mengendalikan hak akses ke data dalam basis data, termasuk memberikan dan mencabut izin.
  4. TCL (Transaction Control Language) digunakan untuk mengelola transaksi dalam basis data, memastikan bahwa operasi dilakukan dengan cara yang aman dan konsisten

MySQL

MySQL adalah salah satu DBMS (Database Management System) paling populer yang menggunakan SQL sebagai bahasa interaksinya. MySQL adalah sistem manajemen basis data relasional open-source yang banyak digunakan untuk aplikasi web dan server. MySQL dikelola oleh Oracle Corporation dan terkenal karena kinerjanya yang cepat, kehandalannya, dan kemudahan penggunaannya.

  • Open-Source: MySQL tersedia secara gratis dengan lisensi open-source, tetapi juga memiliki versi komersial dengan fitur tambahan.
  • Portabilitas: MySQL dapat berjalan di berbagai platform termasuk Windows, Linux, dan macOS.
  • Skalabilitas dan Fleksibilitas: MySQL dapat menangani basis data kecil hingga besar dengan mudah.
  • Keamanan: MySQL menyediakan fitur keamanan yang kuat seperti autentikasi pengguna, enkripsi data, dan kontrol akses.
  • Performa Tinggi: MySQL dioptimalkan untuk memberikan kinerja tinggi dalam penanganan kueri dan transaksi

Menginstall dan Konfigurasi MySQL

Menggunakan MySQL Installer

  1. Buka browser web dan pergi ke situs web resmi MySQL: MySQL Downloads.
  2. Pilih "MySQL Installer for Windows" dan klik "Download".
  3. Buka file installer yang telah kamu unduh (mysql-installer-community-x.x.x.x.msi).
  4. Pilih tipe instalasi:
    • Developer Default: Menginstal semua produk MySQL yang umum digunakan oleh pengembang.
    • Server only: Hanya menginstal MySQL Server.
    • Client only: Hanya menginstal alat-alat client.
    • Full: Menginstal semua produk MySQL.
    • Custom: Memungkinkan kamu untuk memilih produk apa saja yang akan diinstal. Pilih opsi ini jika kamu ingin kontrol lebih atas apa yang akan diinstal.
  5. Pada layar "Select Products and Features", pilih produk yang ingin diinstal. Biasanya, kamu memerlukan MySQL Server dan MySQL Workbench.
  6. Klik "Next" setelah memilih produk.
  7. Installer akan memeriksa apakah ada perangkat lunak tambahan yang diperlukan (seperti Visual Studio Redistributable).
  8. Jika ada perangkat lunak yang diperlukan, installer akan memberikan opsi untuk mengunduh dan menginstalnya. Klik "Execute" untuk melanjutkan.
  9. Pada layar "Installation", klik "Execute" untuk memulai instalasi produk yang dipilih.
  10. Setelah instalasi selesai, klik "Next".
  11. Pada layar "Product Configuration", pilih "MySQL Server" dan klik "Next".
  12. Pilih tipe konfigurasi server:
    • Development Machine: Untuk komputer pengembangan. Menggunakan sumber daya yang lebih sedikit.
    • Server Machine: Untuk server dengan beban kerja sedang.
    • Dedicated Machine: Untuk server yang didedikasikan hanya untuk MySQL. Menggunakan sumber daya maksimal.
  13. Klik "Next".
  14. Pada layar "Type and Networking", pilih tipe koneksi (biasanya, gunakan pengaturan default).
  15. Pada layar "Authentication Method", pilih metode autentikasi (biasanya, gunakan "Use Strong Password Encryption" untuk keamanan lebih baik).
  16. Klik "Next".
  17. Masukkan dan konfirmasi kata sandi untuk pengguna root (administrator).
  18. Jika diinginkan, tambahkan pengguna baru dengan klik "Add User".
  19. Klik "Next".
  20. Pada layar "Windows Service", pilih untuk menjalankan MySQL Server sebagai layanan Windows agar dapat otomatis berjalan saat Windows dihidupkan.
  21. Klik "Next".
  22. Pada layar "Apply Configuration", klik "Execute" untuk menerapkan semua pengaturan dan konfigurasi.
  23. Setelah selesai, klik "Finish".
  24. Pada layar "Installation Complete", klik "Finish".
  25. Buka MySQL Workbench atau MySQL Command Line Client untuk mulai menggunakan MySQL.

Menggunakan XAMPP

  1. Buka browser web dan pergi ke situs web resmi XAMPP: XAMPP Downloads.
  2. Pilih versi XAMPP untuk Windows dan klik "Download".
  3. Buka file installer yang telah kamu unduh (xampp-windows-x.x.x-x-installer.exe).
  4. Jika muncul jendela "User Account Control", klik "Yes" untuk melanjutkan.
  5. Pada layar "Select Components", pastikan bahwa "MySQL" dicentang. Kamu juga bisa mencentang komponen lain seperti "Apache", "PHP", dan "phpMyAdmin" jika diperlukan.
  6. Klik "Next" untuk melanjutkan.
  7. Pilih folder di mana kamu ingin menginstal XAMPP. Secara default, XAMPP akan diinstal di C:\xampp
  8. Klik "Next" untuk melanjutkan.
  9. Pada layar "Ready to Install", klik "Next" untuk memulai proses instalasi.
  10. Tunggu hingga proses instalasi selesai. Ini mungkin memakan waktu beberapa menit.
  11. Setelah instalasi selesai, kamu akan melihat layar konfirmasi. Klik "Finish" untuk menyelesaikan instalasi.
  12. XAMPP Control Panel akan otomatis terbuka.
  13. Di XAMPP Control Panel, kamu akan melihat daftar layanan termasuk "Apache" dan "MySQL".
  14. Klik tombol "Start" di sebelah "MySQL" untuk memulai MySQL Server.
  15. Jika MySQL berhasil dijalankan, statusnya akan berubah menjadi hijau.
  16. Untuk mengelola basis data MySQL, kamu dapat menggunakan phpMyAdmin, alat administrasi basis data berbasis web yang disertakan dengan XAMPP.
  17. Buka browser web dan pergi ke URL: http://localhost/phpmyadmin/
  18. Kamu akan masuk ke antarmuka phpMyAdmin di mana kamu bisa membuat dan mengelola basis data MySQL
  19. Di phpMyAdmin, klik tab "Databases"
  20. Di bagian "Create database", masukkan nama basis data yang ingin kamu buat dan klik "Create".
  21. Setelah basis data dibuat, kamu bisa mengelolanya dengan phpMyAdmin.
  22. Kamu bisa membuat tabel baru, memasukkan data, menjalankan kueri SQL, dan banyak lagi.
  23. Jangan lupa masukkan xampp/mysql/bin ke dalam path di environment variable
  24. Setelah itu, buka cmd dan cek apakah mysql telah terinstall dengan perintah "mysql -v"

Dengan XAMPP, kamu memiliki lingkungan pengembangan web yang lengkap termasuk MySQL. Ini sangat membantu untuk pengembangan lokal dan pengujian aplikasi web sebelum dipublikasikan di server produksi.
