Operasi Matematika

SQL menyediakan berbagai fungsi numerik untuk melakukan operasi aritmatika dan manipulasi angka. Berikut adalah beberapa fungsi numerik yang umum digunakan di SQL beserta penjelasan dan contohnya:

Penjumlahan (+)

Operator + digunakan untuk menambahkan dua angka. 

        SELECT 5 + 3 AS result;

hasilnya adalah

        result
        ------
        8

Pengurangan (-)

Operator - digunakan untuk mengurangi satu angka dari angka lainnya.

        SELECT 10 - 4 AS result;

dengan hasil 

        result
        ------
        6

Perkalian (*)

Operator * digunakan untuk mengalikan dua angka. 

        SELECT 7 * 2 AS result;

yang akan menghasilkan 

        result
        ------
        14

Pembagian (/)

Operator / digunakan untuk membagi satu angka dengan angka lainnya. Hasilnya adalah angka desimal. 

        SELECT 15 / 2 AS result;

Hasilnya 

        result
        ------
        7.5

Pembagian Integer (DIV)

Operator DIV digunakan untuk membagi satu angka dengan angka lainnya dan mengembalikan bagian integer dari hasil pembagian. 

        SELECT 15 DIV 2 AS result;

Hasilnya 

        result
        ------
        7

Modulo (%)

Operator % atau MOD digunakan untuk mendapatkan sisa dari pembagian dua angka. 

        SELECT 15 % 4 AS result;
        -- Atau menggunakan fungsi MOD
        SELECT MOD(15, 4) AS result;

Hasilnya 

        result
        ------
        3

Mengubah Tanda Angka (-)

Operator - dapat digunakan untuk mengubah tanda suatu angka dari positif ke negatif atau sebaliknya. 

        SELECT -(-5) AS result;

Hasilnya 

        result
        ------
        5

Contoh Penggunaan dalam Query

Misalkan kita memiliki tabel products dengan data berikut: 

        +----+------------+-------+
        | id | product    | price |
        +----+------------+-------+
        | 1  | Product A  | 100   |
        | 2  | Product B  | 200   |
        | 3  | Product C  | 300   |
        +----+------------+-------+

Menghitung Diskon 10% untuk Semua Produk 

        SELECT product, price, price * 0.9 AS discounted_price
        FROM products;

Hasilnya

        +-----------+-------+-----------------+
        | product   | price | discounted_price|
        +-----------+-------+-----------------+
        | Product A | 100   | 90.0            |
        | Product B | 200   | 180.0           |
        | Product C | 300   | 270.0           |
        +-----------+-------+-----------------+

Fungsi Lain

Fungsi Deskripsi Contoh Penggunaan Hasil
ABS(x) Mengembalikan nilai absolut dari x SELECT ABS(-5) AS result; 5
CEIL(x) Mengembalikan nilai terkecil yang lebih besar atau sama dengan x (pembulatan ke atas) SELECT CEIL(4.2) AS result; 5
FLOOR(x) Mengembalikan nilai terbesar yang kurang dari atau sama dengan x (pembulatan ke bawah) SELECT FLOOR(4.8) AS result; 4
ROUND(x) Membulatkan x ke bilangan bulat terdekat SELECT ROUND(4.6) AS result; 5
ROUND(x, d) Membulatkan x ke d tempat desimal SELECT ROUND(4.678, 2) AS result; 4.68
POWER(x, y) Mengembalikan x pangkat y SELECT POWER(2, 3) AS result; 8
SQRT(x) Mengembalikan akar kuadrat dari x SELECT SQRT(16) AS result; 4
EXP(x) Mengembalikan e pangkat x SELECT EXP(1) AS result; 2.7182818
LN(x) Mengembalikan logaritma natural dari x SELECT LN(2.7182818) AS result; 1
LOG10(x) Mengembalikan logaritma basis 10 dari x SELECT LOG10(100) AS result; 2
GREATEST(x1, x2, ...) Mengembalikan nilai terbesar dari sekumpulan nilai SELECT GREATEST(3, 7, 5) AS result; 7
LEAST(x1, x2, ...) Mengembalikan nilai terkecil dari sekumpulan nilai SELECT LEAST(3, 7, 5) AS result; 3
